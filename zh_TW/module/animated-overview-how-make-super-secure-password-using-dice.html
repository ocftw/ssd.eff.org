動畫概述：如何使用骰子製作一組超安全的密碼

電腦密碼有解開大量豐富資訊的能力，但有些密碼太過簡單，很容易被小偷猜到。常見的好方法是避免使用常見的語句 (如「iloveyou」)，並添加符號或數字到您的密碼，讓它們很難被猜中。不過，現在的攻擊者會使用電腦程式來猜密碼，每秒能嘗試數百萬個密碼。密碼若要不被攻擊者猜中，需要夠長且隨機。但對一般人而言，又長又是隨機排列的密碼想當然是非常難記憶的。

要設計一個真正隨機而您確實可以記住的密碼，請嘗試使用 Arnold G. Reinhold 的 Diceware 技術，這是一個用骰子和 Diceware 單詞列表建立安全密碼的方法。

由此產生的密碼應該夠隨機，足以抵禦自動密碼破解者（automatic password crakcer） (除了那些擁有大筆預算的大型組織，例如 NSA 或其他諜報機構)。請務必將您的強密碼（strong password）保密，並且避免重複使用。取決於您在密碼中加入多少個隨機詞，您甚至可以阻止最厲害的攻擊者。例如，一般認為五個隨機單詞 (64 位) 有助於防止犯罪分子的攻擊，而六個隨機單詞 (77 位) 有助於防範幾乎所有人，除了最具目的性的國家級攻擊者 (如 NSA) 。

此動畫概述包含有關如何使用 Diceware 的詳細說明；若要進一步了解如何管理多個密碼，請查看以下指引：

  • 建立強式密碼
