動畫概述：如何使用骰子製作一組超安全的密碼

電腦密碼有能力解開大量的資訊，但有些密碼太過簡單，宵小竊賊很容易猜到。一個常見的最佳做法，是避免使用常見的語句 (如「iloveyou」)，並添加符號或數字到您的密碼，提高會被猜中的難度。但是，現在的攻擊者使用電腦程式來猜密碼，而且這些程式每秒都能嘗試數百萬個密碼。密碼要不被這樣的攻擊者猜中，需要長度夠長而且隨機排列。但是一個長而隨機的密碼對於平凡的人類來說是非常難以記憶的。

要設計一個真正隨機而您確實可以記住的密碼，請嘗試使用 Arnold G. Reinhold 的 Diceware 技術，這是一個用骰子和 Diceware 單詞列表建立安全密碼的方法。

由此產生的密碼應該夠隨機，足以抵禦自動密碼破解 (除了那些擁有大筆預算的大型組織，例如 NSA 或其他諜報機構)。務必要把您的強力密碼保密，而且避免重複使用密碼。取決於您在密碼中加入多少個隨機詞，您甚至可以阻止最厲害的攻擊者。例如，一般認為五個隨機單詞 (64 位) 有助於防止犯罪分子的攻擊，而六個隨機單詞 (77 位) 有助於防範幾乎所有人，除了最積極的國家級攻擊者 (如 NSA) 以外。

此動畫概述包含有關如何使用 Diceware 的詳細說明；要進一步了解如何管理多個密碼，請查看以下指引：
