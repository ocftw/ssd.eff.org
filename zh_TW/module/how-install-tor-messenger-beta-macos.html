如何：在 macOS 上安裝 Tor Messenger (beta)

下載位置：https://blog.torproject.org/blog/tor-messenger-beta-chat-over-tor-easily

電腦需求：網際網路連線，運行 macOS 的電腦

本指引中使用的版本：macOS Sierra (10.12.3)

授權：自由軟體；各種自由軟體授權

級別：初學者

所需時間：10 分鐘

什麼是 Tor Messenger？Anchor link

Tor Messenger 是一個即時通訊用戶端程式，藉由 Tor 匿名網路發送訊息。它可在 Linux、macOS 和 Windows 使用。Tor Messenger 支援各種通訊網路和協定，包括 IRC、XMPP、Twitter、Google Talk 等。除此之外，還有內建不記錄 (OTR) 加密。

Tor Messenger 的預設配置僅使用加密的不記錄 (OTR) 協定進行通訊。這表示您沒有辦法跟一個沒有支援 OTR 通訊程式的人聊天。您可以透過進入「工具 (Tools) -> OTR 偏好設定 (OTR Preferences)」並取消勾選「需要加密 (Require encryption)」來更改此選項。但請記得，這會讓對話以明文傳送，而讓即時通訊提供者看到訊息內容。

Tor Messenger 仍處於測試階段，這表示它的可靠程度或功能可能仍待改進。請謹慎使用。

下載並安裝用戶端 Anchor link

請前往 https://blog.torproject.org/tor-messenger-beta-chat-over-tor-easily。點擊「取得最新版本 (Get the latest version」連結。在這個頁面上，點擊「macOS」連結。

Tor Messenger 安裝程式將下載到您的下載檔案夾中。點擊桌面上的「下載 (Downloads)」圖示即可。

然後點擊安裝檔案。

畫面上將出現一個視窗，讓您安裝 Tor Messenger 到您的應用程式中。將 Tor Messenger 聊天氣泡拖曳到 Applications 檔案夾進行安裝。

現在您如果在 Finder 中輸入「tormessenger」，這將是第一筆結果。

在綠色的 Tor Messenger 圖示點兩下，啟動應用程式。可能會彈出一個警告視窗，告訴您這是從網際網路下載的應用程式。您可以安心點擊「開啟 (Open)」。

應用程式啟動時，您可以長按圖示，選擇「選項 (options)」，然後選擇「Keep in Dock (保留在 Dock 中)」，讓它保持在 Dock 中。

現在您已經準備好設定程式並開始聊天。請參閱我們的如何：在 macOS 上使用 Tor Messenger (beta) 的逐步說明。
