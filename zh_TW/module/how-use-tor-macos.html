如何：在 macOS 上使用 Tor

本指引簡略介紹如何在 macOS 上使用 Tor 瀏覽器。

電腦需求：網際網路連線，運行近期版本 macOS 的電腦

本指引中使用的版本：macOS 10.12.6；Tor 瀏覽器 7.0.2

授權：自由軟體；各種自由軟體授權

其他閱讀資訊：

  • https://www.torproject.org/docs/documentation.html.en
  • https://www.eff.org/pages/tor-and-https
  • https://tor.stackexchange.com/

級別：初級至中級

所需時間：15-30 分鐘

什麼是 Tor？Anchor link

Tor 是一項由志工運作的服務，藉由掩蓋您的身分和連線目的地，提供線上隱私和匿名服務。該服務還可以保護您不受 Tor 網路本身侵害：確保您對其他 Tor 使用者保持匿名。

對於在瀏覽網站時可能需要偶爾保持匿名和保護隱私的人，Tor 瀏覽器提供了一種快速、便捷的方式來使用 Tor 網路。

Tor 瀏覽器的運作方式就像網頁瀏覽器一樣。網頁瀏覽器是您用來瀏覽網站的程式，像是 Chrome、Firefox 和 Safari。與其他瀏覽器不同，Tor 瀏覽器使用 Tor 網路發送您的通訊，這讓監控您的人很難知道您在網路上做了些什麼，而監控您所瀏覽的網站的人也難以得知您是從哪裡連線的。

請記得，只有在 Tor 瀏覽器內進行的活動才會匿名。在您的電腦上安裝 Tor 瀏覽器，並不會讓您在同一部電腦上使用其他軟體 (例如您的一般網頁瀏覽器) 時匿名。

取得 Tor 瀏覽器 Anchor link

打開 Firefox 或 Safari 等瀏覽器，然後在網址列輸入：

https://www.torproject.org/download/download-easy.html.en。如果您是使用搜尋引擎來查找 Tor 瀏覽器，請確認 URL 正確。

不要使用任何其他來源，並且如果系統提示您接受其他 HTTPS (SSL/TLS) 安全性憑證，請勿繼續。

點擊大型的紫色下載按鈕，取得 Tor 瀏覽器的安裝程式。

該網站會檢測您的作業系統，讓您取得適用於 macOS 的檔案。如果紫色按鈕失效，您可以點擊按鈕旁邊的連結，下載正確的版本。

如果您使用 Safari，將開始下載 Tor 瀏覽器。在 Firefox 中，會詢問您是否要開啟或儲存檔案。儲存檔案是最佳選擇，所以請點擊儲存 (Save) 按鈕。此範例畫面為 Tor 瀏覽器 7.0.2 版，是本指引發布時的最新版本。在您閱讀本文時，可能會有更新的版本可供下載。

安裝 Tor 瀏覽器 Anchor link

下載完成後，您可以選擇打開下載檔案的檔案夾。預設位置是 Downloads 檔案夾。在 TorBrowser-7.0.2-osx64_en-US.dmg 檔案上點兩下。

MacOS 會驗證應用程式有沒有被竄改。

接下來會打開一個視窗，要求您將 Tor 瀏覽器拖曳到應用程式檔案夾中。您現在可以這樣做。

Tor 瀏覽器現在已安裝在您的應用程式檔案夾。

使用 Tor 瀏覽器 Anchor link

首次開啟 Tor 瀏覽器，請在應用程式檔案夾或 Launchpad 中找到它。

點擊 Tor 瀏覽器圖示後，會打開一個視窗，警告軟體的來源。您應該認真看待這些警告，確保您信任所要安裝的軟體，並且確保它是以安全連線從官方網站下載的正確檔案。如果您已經知道您所要安裝的檔案是什麼，也瞭解軟體的來源，該檔案也是從 Tor 計畫的安全 HTTPS 網站下載的，請點擊開啟 (Open)。

Tor 瀏覽器首次啟動時，您會看到一個視窗，讓您視需要修改某些設定。您可能必須回來更改一些配置，但請先點擊「連線」(Connect) 按鈕連接 Tor 網路。

點擊連線 (Connect) 後，會開啟一個帶有綠色進度條的新視窗，並隨著 Tor 軟體啟動變得更長。

Tor 瀏覽器首次啟動時，可能需要一段時間；請耐心等待，在一、兩分鐘內，Tor 瀏覽器就會打開並恭喜您。

點擊 Tor 瀏覽器左上方的 Tor Onion 圖示，然後點擊隱私和安全設置 (Privacy and Security Settings)。

普通網頁瀏覽器有些功能可能會讓您容易受到攔截式攻擊，有些則存在會洩漏使用者的身分的臭蟲。為了停用瀏覽器這些不安全的功能，請將安全滑桿調到最高的安全層級。這可讓您安全防範有些資金充沛，透過這些功能以干擾您網路連線，或利用新的未知臭蟲來攻擊您的攻擊者。雖然預設的低安全層級足以保護您日常的網路隱私，但是如果您擔心受到訓練精良的攻擊者侵入，或是您並不介意某些網站顯示不正確，您可以將安全設置調高。

最後，使用 Tor 瀏覽器與使用一般瀏覽器有所不同。我們建議您先閱讀這些技巧，以正確使用 Tor 瀏覽並保有匿名性。

現在您已經準備好用 Tor 匿名瀏覽網際網路了。
