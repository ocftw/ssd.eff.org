如何：在 macOS 上使用 Tor Messenger (beta)

下載位置：https://blog.torproject.org/blog/tor-messenger-beta-chat-over-tor-easily

電腦需求：網際網路連線，運行 macOS 的電腦

本指引中使用的版本：macOS Sierra (10.12.3)

授權：自由軟體；各種自由軟體授權

 級別：初學者

所需時間：10 分鐘

什麼是 Tor Messenger？Anchor link

Tor Messenger 是一個即時通訊用戶端程式，藉由 Tor 匿名網路發送訊息。它可在 Linux、macOS 和 Windows 使用。Tor Messenger 支援各種通訊網路和協定，包括 IRC、XMPP、Twitter、Google Talk 等。除此之外，還有內建不記錄 (OTR) 加密。

Tor Messenger 的預設配置僅使用加密的不記錄 (OTR) 協定進行通訊。這表示您沒有辦法跟一個沒有支援 OTR 通訊程式的人聊天。您可以透過進入「工具 (Tools) -> OTR 偏好設定 (OTR Preferences)」並取消勾選「需要加密 (Require encryption)」來更改此選項。但請記得，這會讓對話以明文傳送，而讓即時通訊提供者看到訊息內容。

Tor Messenger 仍處於測試階段，這表示它的可靠程度或功能可能仍待改進。請謹慎使用。

下載並安裝用戶端 Anchor link

在繼續之前，請安裝 Tor Messenger。請參閱我們的如何：在 macOS 上安裝 Tor Messenger (beta) 的逐步說明。

連線到 Tor Anchor link

第一次執行 Tor Messenger 時，您會看到以下畫面：

在大多數情況下，您可以選擇第一個選項，然後按一下「連線」(Connect)。如果您的網路環境會被審查或使用代理伺服器（例如，您所在國家的政府會過濾審查網路)，請選擇第二個選項「配置」(Configure) ，它將引導您完成連線到 Tor 的步驟。

連線到 Tor 後，所有的 Tor Messenger 通訊都會透過 Tor 網路傳輸。這表示您正在使用的服務將無法取得您的位置。如果您在註冊 Tor Messenger 帳戶時，選擇隨機的使用者名稱，並只在 Tor Messenger 上使用，也可達到不錯的匿名性效果。

 

新增帳戶 Anchor link

您現在會收到新增帳戶的提示：

您可以從這幾個聊天系統中，選出一個聊天系統使用。在本指引，我們將著重介紹 XMPP (以前稱為 Jabber)。它可讓您與其他使用 XMPP 的人通訊。同樣地，選擇 Twitter、Google Talk、IRC 或其他系統，也可讓您與其他使用該系統的人通訊。但 Gchat 用戶必須注意，他們將無法與其他 XMPP 伺服器上的使用者聊天。點擊 XMPP，然後點擊「繼續」(Continue)。

就像您選擇電子郵件提供者一樣，您也可以選擇 XMPP 的提供者。選哪一個都可以，像 jabber.calyxinstitute.org 就是一個蠻不錯的伺服器。請為您的帳戶選擇一個用戶名以及要連線的伺服器。如果您不是要登入現有帳戶，而是要建立新帳戶，請勾選「在伺服器上建立此新帳戶」(Create this new account on the server) 方塊。

接下來，選擇一個密碼。更多技巧請參閱我們的建立強式密碼指引。

點擊繼續，如果需要的話，可輸入一個別名，然後再次點擊繼續。這個別名可以是比您的使用者名稱和網域名稱更短、更容易記住的綽號或假名。在下面的畫面中，您可以選擇每次 Tor Messenger 啟動時，都連線到該聊天帳戶。點擊「完成」(Done)，然後點擊「連線」(Connect)。

再次輸入您的密碼，然後點擊「確定」(OK)。您現在會連線到 XMPP 伺服器。

新增聯絡人 Anchor link

從 Tor Messenger 的主視窗，點擊「檔案」(File)，然後點擊「新增聯絡人」(Add Contact)。以「 username@domain.tld」的格式輸入您要新增聯絡人的使用者名稱。然後點擊「確定」(OK)。

聯絡人同意您的邀請後，只要他們登入，就會出現在您的聯絡人列表中。同樣地，如果其他人希望將您加入聯絡人列表，也必須經過您同意。

 

聊天 Anchor link

要開始聊天，請在聯絡人上點兩下，然後輸入您的訊息。

如果您的聯絡人也支援不記錄 (OTR) 通訊，您的通訊用戶端將在您傳送訊息給他們時開始加密聊天。請注意聊天視窗右上角的上鎖圖示，以確認加密正在運作。

驗證聯絡人 Anchor link

這時您可以驗證聯絡人的身分。這個過程稱為金鑰驗證。您可以藉此確保他們的加密金鑰沒有被竄改，或在您的應用程式下載金鑰時被換成別人的。這個過程很重要，因為它可確保您的通訊對象確實是您所想聯絡的那個人。

在聊天視窗中，點擊鎖定圖示，然後點擊「驗證聯絡人身分」(Verify your contact’s identity)。

您會看到各種驗證方法的列表。大多數用戶端支援「手動指紋驗證」，它會是一個不錯的選擇。

要進行手動指紋驗證，請向您的聯絡人大聲讀出帳戶的指紋 (「您的指紋」[Fingerprint for you] 後面的一長串數字)。請您的聯絡人也做同樣的事情，並確保它與「聲稱的指紋...」(Purported fingerprint for...) 之後的長串數字相符。最好是面對面或透過電話進行此步驟。無論如何，除了您要驗證的帳戶之外，請務必透過其他管道交換指紋。

一旦您確認每個帳戶都有相同的指紋，請將「我已驗證這個指紋確實正確」(I have verified that this is in fact the correct fingerprint) 的方塊更改為「是」(Yes)，然後點擊「驗證」(Verify)。

聊天視窗右上角的鎖頭現在應該會從橘色變為綠色。這表示您的聯絡人已經驗證完成。如果您的聯絡人重新安裝他們的聊天用戶端，您必須再次驗證指紋以確認他們的身分。

現在您已經知道如何使用 Tor Messenger，請嘗試使用此即時通訊用戶端與您的朋友進行加密聊天。
