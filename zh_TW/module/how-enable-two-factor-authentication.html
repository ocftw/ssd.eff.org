如何：啟用雙因素驗證

雙因素驗證 (或「2FA」) 是以要求兩種不同身分驗證方法的組合，讓使用者向服務提供者識別其身分的方法。這些方法可能是使用者知道的東西 (如密碼或 PIN)、使用者擁有的東西 (如硬體 token 或行動電話)，或與使用者相關或不可分離的東西 (如指紋)。

您可能已經在生活中的其他場合使用過 2FA。您使用 ATM 提款時，必須同時擁有實體的提款卡 (你擁有的東西) 和密碼 (您知道的東西)。但是，現在許多線上服務預設只使用一個因素來識別他們的使用者，也就是密碼。

2FA 如何線上運作？Anchor link

包括 Facebook、Google 和 Twitter 在內的一些線上服務提供 2FA 作為只用密碼驗證身分的替代方案。如果啟用此功能，系統將提示您輸入密碼和輔助的驗證方法。第二種方法通常是由簡訊發送的單次代碼，或由存有祕密的專用行動應用程式 (例如 Google Authenticator、Duo Mobile、Facebook 應用程式或 Clef) 產生的單次代碼。在這兩種情況下，第二個因素是您的手機，您 (通常) 擁有的東西。某些網站 (包括 Google) 也支援單次使用的備用代碼，可以下載、印在紙上，存放在安全的位置做為備份。選擇使用 2FA 後，您需要輸入密碼和手機上的單次代碼，才能存取您的帳戶。

為什麼要啟用 2FA？Anchor link

2FA 藉由要求您使用多種方法驗證您的身分，提供更高的帳戶安全性。這也就表示，即使有人取得您的主密碼，也無法存取您的帳戶，除非他們也有您的手機或其他輔助認證方法。

使用 2FA 有缺點嗎？Anchor link

儘管 2FA 提供了更安全的身分驗證方式，但是如果您遺忘或丟失了手機、更換 SIM 卡，或是在不開啟漫遊的情況下前往其他國家，很有可能無法登入帳戶。

許多 2FA 服務提供單次使用的「備份」或「回復」代碼列表。每個代碼只能用來登入您的帳戶一次，之後就會失效。如果您擔心無法取得您的手機或其他身分驗證裝置，請印出並隨身攜帶這些代碼。他們的效果就像「您擁有的東西」一樣，只要您只印一份，並加以保密。請記得妥善保管這些代碼，並隨時確保沒有人可以看到或拿到。如果您使用或遺失了備用驗證碼，可以在下次登入帳戶時產生新的列表。

使用簡訊的 2FA 系統有另一個問題，就是簡訊不夠安全。可以存取電話網路的高階攻擊者 (例如情報機關或組織犯罪行動)，可能會攔截並使用由簡訊發送的代碼。還有一些情況是，不那麼高階的攻擊者 (例如個人) 設法將打給一個號碼的電話或簡訊轉發給自己，或者存取電話公司的服務，這些服務可以顯示發送到電話號碼的簡訊，而不用持有電話。

如果您擔心這種層級的攻擊，可以關閉簡訊驗證，只使用 Google Authenticator 或 Authy 等身分驗證應用程式。然而這個選項不適用於每一種使用 2FA 的服務。

另外，使用 2FA 表示您可能必須把比您所願意還要更多的資訊交給服務。假設您使用 Twitter，並使用假名註冊。即使您小心避免向 Twitter 提供您的身分資訊，即使您只藉由 Tor 或 VPN 存取該服務，如果您啟用了簡訊 2FA，則 Twitter 必須記錄您的手機號碼。這表示如果法院強制要求，Twitter 可以藉由您的電話號碼將您的帳戶與您連結。這對您來說可能不是問題，特別是如果您已經在現有服務上使用了您的合法名稱，但如果保持匿名對您來說很重要的話，請謹慎考慮要不要使用簡訊 2FA。

最後，研究顯示，啟用 2FA 之後，一些使用者會使用強度更低的密碼，認為第二個因素會保障他們的安全。即使啟用 2FA 之後，務必仍然要使用強式密碼。請參閱我們的建立強式密碼指引以獲取提示。

我要如何啟用 2FA？Anchor link

這在每一個平台不同，所使用的術語也不同。https://twofactorauth.org/ 提供了支援 2FA 的大量網站列表。對於最常見的服務，您可以參考我們的 12 Days of 2FA 貼文，說明如何在 Amazon、Bank of America、Dropbox、Facebook、Gmail 和 Google、LinkedIn、Outlook.com 和 Microsoft、PayPal、Slack、Twitter 和 Yahoo Mail 啟用 2FA。

如果您想要進一步防範密碼竊取，請仔細閱讀這一系列文章，並為您仰賴的所有重要網路帳戶開啟 2FA。
