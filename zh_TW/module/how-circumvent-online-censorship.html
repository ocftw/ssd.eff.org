如何：繞過線上審查

這是一個繞過線上審查的簡短概述，內容絕不全面。

有時，政府、公司、學校和網際網路提供商會使用軟體來阻止他們的使用者存取某些網站和服務，這就是所謂的網際網路過濾或封鎖，是一種網路審查的形式。過濾有多種類型，例如：審查人員可封鎖個別的網頁，甚至整個網站，有時則是根據內容所包含的關鍵字進行封鎖。

抵制網際網路審查的方法有許多種。有些方法可讓您不受監控，但大多數無法辦到。當有人控制您的網路連線過濾器或封鎖網站時，您幾乎完全可使用規避工具來獲取需要的資訊。但請注意：那些號稱可保護隱私或安全的規避工具，未必能完全保障隱私或安全。而使用像「匿名者」（“anonymizer”）這種術語的工具，也未必能完全保密您的身分。

最適合您的規避工具取決於您的威脅模型。如果您不確定您的威脅模型是什麼，請從這裡開始。

在這篇文章中，我們將談到四種繞過審查制度的方法：

  • 前往網頁式代理伺服器，存取被封鎖的網站。
  • 前往加密的網頁式代理伺服器，存取被封鎖的網站。
  • 使用虛擬私人網路 (VPN) 存取被封鎖的網站或服務。
  • 使用 Tor 瀏覽器存取被封鎖的網站或保護您的身分。

基本技巧 Anchor link

規避工具通常是以分流網路流量的方式，避開進行封鎖或過濾的設備。那些將您的網路連線服務重新導向、繞過封鎖的服務，有時稱為代理伺服器 (proxy)。

HTTPS 是用來存取網站的 HTTP 協定的加密版本。有時候審查人員只會封鎖網站的未加密 (HTTP) 版本。這表示只要輸入以 HTTPS 開頭的網址，即可存取被封鎖的網站。

如果您正在對抗的審查制度是依據個別網頁的內容進行封鎖，這個方法會非常有用。由於 HTTPS 會防止審查人員讀取您的網路流量，他們便無法知道您傳送了什麼關鍵字，或瀏覽了哪個網頁。

審查人員仍可查看所有您瀏覽的網站的網域名稱。因此，假如您存取 "eff.org/https-everywhere"，審查人員可以看到您在 "eff.org"，但看不到您是在 "https-everywhere" 頁面。

如果您懷疑有此類簡單的封鎖，請嘗試輸入在網域名稱之前輸入 https:// 取代 http:

請嘗試安裝 EFF 的 HTTPS Everywhere 擴充功能，以盡可能自動使用 HTTPS 連線。

另外一種可以繞過基本審查技術的方法，是嘗試使用替代網域名稱或 URL。例如，當您要瀏覽 http://twitter.com，可以嘗試使用行動版網站 http://m.twitter.com。由於審查人員會從被禁網站的黑名單上封鎖網站或網頁，所以只要是不在黑名單上的網站或網頁，就可繞過審查。審查人員可能不知道特定網站名稱的所有版本，特別是如果網站管理者知道網站遭到封鎖，而註冊了不只一個網域名稱。

網頁式代理伺服器 Anchor link

網頁式代理伺服器 (如 http://proxy.org/) 是一個網站，可讓使用者存取其他被封鎖或審查的網站。它可以有效地繞過審查制度。若要使用網頁式代理伺服器，請前往代理網頁並輸入您想要查看的網址，代理伺服器將顯示您要求的網頁。

但是，網頁式代理伺服器不提供任何加密，如果您的威脅模型包括有人監控您的網際網路連線，它便不是個好選擇。它不會幫助您使用被封鎖的服務，例如您的即時通訊應用程式。網頁式代理伺服器會完整記錄您在網路上做的所有事情，取決於使用者的威脅模型，這對於一些使用者來說可能是一種隱私風險。

加密的代理伺服器 Anchor link

許多代理伺服器工具除了具備繞過過濾的能力，還會善加利用加密，讓服務更添一層安全性。由於是加密連線，所以其他人無法看到您在瀏覽的內容。雖然加密代理伺服器通常比普通的網頁式代理伺服器更安全，但是工具提供者可能會取得有關您的資訊。例如，他們可能會在紀錄中得到您的姓名和電子郵件位址。這也意味著，這些工具並非完全匿名。

最簡單的加密網頁式代理伺服器的形式是以「https」開頭——它會使用許多安全網站常提供的加密。但請小心，這些代理伺服器（例如 Ultrasurf 和 Psiphon）的擁有者可看到您發送到其他安全網站、收到來自其他安全網站的資料。

虛擬私人網路 Anchor link

虛擬私人網路 (VPN) 會將您電腦上所有的網際網路資料，加密並傳送至另一部電腦。這部電腦可能屬於商用或非營利 VPN 服務、您的公司或是信任的聯絡人。只要 VPN 服務設置正確，您就可以使用它來存取網頁、電子郵件、即時通訊、使用 VoIP 和其他任何網際網路服務。VPN 可保護您的流量不受本地監視，但 VPN 提供者仍可記錄您存取的網站，甚至讓第三方直接窺探您上網。取決於您的威脅模型，政府監聽 VPN 連線或取得 VPN 記錄，可能對您構成極大的風險。對某些使用者來說，這樣的風險可能會超過使用 VPN 的短期效益。

有關特定 VPN 服務的資訊，請按一下這裡。

EFF 並不為各種 VPN 的評價背書。有些具有極佳隱私政策的 VPN ，可能是由有心人士運作。不要使用您不信任的 VPN。

Tor Anchor link

Tor 是一種開源軟體，設計目的在於提供您在網路上的匿名性。Tor 瀏覽器是一個內建於 Tor 匿名網路的網頁瀏覽器。由於 Tor 傳輸網頁瀏覽流量的方式，它也可讓您繞過審查制度。(請參閱我們的如何：在 Linux、macOS 和 Windows 使用 Tor 指引)。

第一次啟動 Tor 瀏覽器時，您可選擇選項「您目前在一個受到審查的網路上」：

Tor 不僅可以繞過幾乎所有的國內審查制度，而且如果設置得當，也可以保護你的身分，不會落入監聽您國家網路的敵人手裡。但是，它的速度可能會很慢，而且難以使用。

要了解如何在桌上型電腦上使用 Tor，使用 Linux 請點擊這裡，macOS 請點擊這裡，使用 Windows 則請點這裡，但是請務必在上面顯示的視窗中點擊「配置」(Configure) 而不是「連線」(Connect)。
