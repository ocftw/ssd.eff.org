Facebook 社團：減低風險

Facebook 社團可以把有共同興趣的人聚集在一起，進行交流、分享新知及進行協同合作計畫。有許多不同種類的社團，用途從籌辦小說粉絲聚會到傳播政治集會或訴求都有。Facebook 社團並不是為安全協同合作而設計的，但是隨著 Facebook 的普及，社團功能不可避免地被許多人用來協同合作，而可能受到他人、惡意 Facebook 使用者或政府的破壞或監控。

在社群媒體平台的早期，Facebook 有時會無預警刪除大型的政治活躍社團，而且該公司以不明確的方式改變隱私設定而聞名。如果您正在討論敏感問題，最好考慮使用其他優先考量安全和隱私的工具或網站。

如果您的群眾不願意或無法離開 Facebook，這建議可能就有點不切實際。因此，如果您必須為涉及敏感話題或易受攻擊的社群建立 Facebook 社團或擔任管理員，那麼您應該考慮幾件事情。

調整您社團的隱私設定 Anchor link

建立社團之前，請考慮一下您的用途和目標。您是否打算用社團來討論具爭議性的話題？發起政治運動？您希望向誰宣傳您的社團？社團成員是否希望他們的成員身分保密？對誰保密？這些考量將幫助您確定您的威脅模型，以及哪個隱私設置最適合您。

Facebook 社團與用於公開代表品牌、企業、組織或公眾人物的 粉絲專頁不同，並不一定是公開的 (Facebook 上的任何人都可以看到)。當您建立社團時，有三種隱私設定可以選擇 - 公開、不公開或私密。根據所選的隱私設定，此圖表顯示誰可以加入這些社團，以及哪些人可以看到它們。

如果您確定 Facebook 粉絲專頁更適合您，請記住粉絲專頁是公共空間，即使沒有 Facebook 帳戶的人也可以看到它們。根據 Facebook 的說法，「如果您按一個粉絲專頁讚，關於您按讚粉絲專頁的動態將出現在您的動態時報上，也可能出現在動態消息中。您可能會顯示在您按讚的粉絲專頁，或有關該粉絲專頁的廣告中。」

Facebook 上的任何人都可以看到公開的 Facebook 社團，包括不友善的使用者或政府人士。而公開和不公開 Facebook 社團都可以搜尋得到。請記住，如果您的 Facebook 社團用於政治目的，這一點尤其重要。

如果您已經建立了一個社團，並希望調整其隱私設定，則該社團的所有管理員都可以更改設定。但是，擁有 5,000 名或更多會員的社團隱私只能更改為更嚴格的設定 (例如：公開改為不公開，不公開改為私密)，以避免社團成員與未預期的使用者分享貼文。如果您決定將您的社團隱私更改為更嚴格的設定，只有 24 小時可以反悔。無論社團的大小，隱私設定更改時，所有成員都會收到通知。

無論您社團的隱私設定有多嚴格，Facebook 都可以存取發佈到其平台的所有內容。他們可能會收到命令，要求將內容交給執法部門。此外，使用者可以舉報或標示社團內的內容，即使社團設定為私密。如果違反社群守則，舉報的內容可能會遭到刪除，使用者則可能會收到對內容違規的臨時禁令。

請注意，根據 Facebook 的說法，如果內容不違反 Facebook 的社群守則，則惡意標示不會導致內容遭到移除，但是不慎誤刪仍然會發生。Facebook 也可能接獲法院命令，被迫交出社團成員名單。

建立社團規則 Anchor link

請記住，與其他任何線上內容一樣，任何能存取您社團的人都可以複製社團裡的對話，或者將內容擷圖並公開分享。雖然社團管理員可以在社團簡介中規定禁止螢幕擷圖，並告訴社團成員，分享螢幕擷圖的成員會逐出社團，但沒有任何技術方法可防止這種類型的資訊洩漏。

您可以考慮為社團建立其他規則或準則，鼓勵正面參與，並有助於保護社團成員的隱私。雖然社團規則可能難以 (甚至無法) 執行，但有助於確定社團的目的，並確定社團內能進行哪些對話。您的成員應該知道，除了您在社團內建立的規則之外，他們還要遵守 Facebook 的社群守則和使用條款。請記住，社團成員可能會公然無視規則，因此為了確定哪些安全解決方案最適合您的社團，我們建議您進行威脅模型評估。請查看我們對於威脅模型的指引，獲得更多資訊。

了解您社團的管理員和版主 Anchor link

管理員對社團的隱私設定和成員擁有很大的權力。只有社團管理員可以指定其他社團成員擔任管理員。管理員可以更改社團設置、管理內容和成員。每個社團可能有不只一位管理員，因此了解誰擔任這些角色非常重要。管理員與版主不同。版主可以管理內容和成員，但不能更改社團設置。點擊這裡了解如何解除管理員或版主的職務。

管理員可以在所有公開、不公開和私密社團的社團設置中啟用成員加入審核功能。這需要管理員核准加入社團的每個新人。

如果管理員將某人加到「公開」或「不公開」社團，則該個人網路中的聯絡人可能會在動態消息或搜尋看到他們已被邀請或加入社團。在一個人不希望別人知道他們與您的社團聯繫，或者他們這樣做是非法的情況下，這一點很重要。因此，您可能需要考慮將社團隱私設置為「私密」。

如果社團管理員允許，社團成員可以將他們的朋友加入社團。使用者被加入社團時，無法做出選擇。這表示有人可能會惡意地將您加入一個誹謗的社團 (「暗中策劃顛覆政府的頭痛人物社團」)。您可以隨時離開任何社團。

關於 Facebook 實名政策以及管理員匿名性的說明 Anchor link

Facebook 不允許使用假名。使用者只能使用「真實身分」，也就是他們的朋友平常稱呼的名字，在可接受的身分證件上所顯示的。雖然社團管理員通常有充分的理由希望保護身分，使用假名的社團管理員帳戶可能會因為違反 Facebook 的實名政策而遭到檢舉後停用。如果發生這種情況，並且社團中沒有管理員，Facebook 將檢查社團裡是否還有版主。如果有的話，所有當前的版主都會被徵求擔任管理員的職務，直到有人接受為止。如果社團也沒有版主，則所有社團成員都會收到「讓我成為管理員」或「建議管理員」的選項。考慮至少讓一個管理員的名字是已知的：可能是一個可以安全地將其身分與社團連結起來的人。

封鎖不受歡迎的使用者 Anchor link

您可能有充分的理由封鎖社團成員。也許他們是違反社團規則的社群成員，或者是得以加入這個社團的外人。只有管​​理員才能從社團驅逐或封鎖某人。希望先前成員不能再看到社團的管理員，應封鎖該使用者。遭到社團管理員封鎖的成員不能再看到該社團或任何相關資訊。請查看這張圖表了解更多資訊。

以前自願離開社團的成員可能仍可以存取其中的一些資訊，例如社團名稱、說明和標籤。例如，一個私密社團的前成員仍然可以搜尋到該社團，查看社團簡介，並查看社團標籤。

了解 Facebook 上的內容遭到刪除時會發生什麼事

Facebook 保留刪除違反其 (非常廣泛) 服務條款的 Facebook 社團之權利。如果您的社團發生這種情況，您不但可能會失去社團成員之前的資訊和討論，還可能無法存取您的成員名單，這表示除非您另外追蹤成員，否則您將無法在社團遭到刪除後重新聯絡您的支持者或社群。

對於 Facebook 從政府、執法部門和個人那裡收到的移除要求，我們還有很多不了解之處；但是我們知道，這樣的要求往往是政治性的，特別是在世界各地，言論自由和集會結社自由時常不受尊重。

您也可以選擇蓄意刪除一個社團。社團建立者可以驅逐其所有成員，然後將社團刪除。刪除社團是永久行為，無法逆轉。管理員不能刪除不是他們建立的社團，除非建立者選擇先離開社團。然而，管理員可以將社團封存。封存社團表示它不會出現在非成員的搜尋結果中，也不會有新成員加入該社團。社團可以由任何管理員取消封存。要詳細了解封存和刪除社團之間的差異，請點擊這裡。

有關 Facebook 刪除內容的詳細資訊，請參閱 Facebook 的隱私政策。即使您已經刪除了資料，Facebook 仍然可以存取 - 特別是如果執法機構要求保留資料。Facebook 執法機關相關資訊 (截至 2017 年 1 月 20 日) 指出：「除非我們在使用者從服務中刪除該內容之前收到有效的保留請求，否則我們不會為執法目的保留資料。」

考慮到這些因素，您現在可以做出明智的決定，確定 Facebook 社團是否適合您的需求。
