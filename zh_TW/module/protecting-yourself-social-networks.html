在社群網路上保護自己

社群網站是網際網路上最熱門的網站及工具之一。Facebook、Google+ 和 Twitter 都有數億使用者。

社群網路通常以分享貼文、照片及個人資訊為基礎，不過它們也成為集會與發表言論的論壇，其中許多都仰賴隱私及化名（pseudonymity）。因此，在使用社群網路時，請思考以下重要問題：我要如何在這些網站與其他使用者互動時保護自己？我的基本隱私？我的身分？我的聯絡人和人際關係？我要保密哪些資訊，不讓哪些人知道這些資訊？

根據您的情況，您可能需要保護自己，不受社群媒體網站本身、網站其他使用者，或者兩者的侵害。

在設定帳戶時，請注意以下幾點提示：

註冊社群媒體網站 Anchor link

  • 您想使用真名嗎？一些社群媒體網站有所謂的「實名制政策」，但隨著時間過去，這些網站變得寬鬆許多。如果您在註冊社群媒體網站時不想使用真實姓名，就不要用。
  • 註冊時，不要提供不必要的資訊。如果您對於隱藏身分有所顧慮，請使用其他的電子郵件位址。注意您的 IP 位址可能會在註冊時被記錄下來。
  • 選擇一個強密碼，並盡可能啟用雙因素驗證。請參考我們的指引，啟用雙因素驗證。
  • 注意，有些密碼回復問題的答案可能可以從您在社群媒體的資訊找到，例如：「您出生在哪個城市？」或「您寵物的名字？」您可能會想在設定密碼回復問題時填入錯誤的答案。記住密碼回復問題的答案有個好辦法，如果您選擇使用錯誤答案來增加安全性，則應在密碼保險箱記錄您所選的答案。

檢查社群媒體網站的隱私政策 Anchor link

請記得，由第三方儲存的資訊受到其自有的政策規範，可能會用於商業目的或提供給其他公司，例如行銷公司。我們都知道，閱讀隱私政策幾乎是不可能的任務，但您可能需要看看如何使用您的資料、何時提供資料給他人，以及服務如何回應執法要求的部分。

社群網站通常是盈利事業，會蒐集超出您明確輸入的敏感資訊：您的所在位置、您有哪些興趣和對哪些廣告做出了反應，還訪問過哪些其他網站 (例如，藉由「讚」按鈕)。封鎖第三方 cookie 並使用封鎖追蹤的瀏覽器擴充功能，確保多餘的資訊不會被動地傳輸給第三方，可能會有幫助。

一些像是 Facebook 和 Twitter 等社群網站，為了更有效地投放廣告，與資料仲介商之間存在一些業務關係。EFF 提供指引，教您如何退出這些追蹤計畫：

  • 如何退出 Facebook 與資料仲介商關係
  • 如何退出 Twitter 與資料仲介商關係

更改您的隱私設定 Anchor link

具體來說，就是更改預設設定。比如說，您想公開分享您的貼文，還是只與特定的一群人分享？其他人能夠用您的電子郵件位址或電話號碼找到您嗎？您想自動分享您的位置嗎？

  • 如何更改您的 Facebook 隱私設定

請記得，隱私設定可能會改變。有時候這些隱私設定變得更強、更詳細，有時候則不然。務必密切注意這些變化，看看以往的私密資訊是否被分享出去，或者是否有任何附加設定可讓您對自己的隱私有更多控制權。

您的社群圖譜 Anchor link

請記得，您不是唯一會洩漏可能為敏感資料的人。您的朋友可以在照片中標記您、報告您的位置，並以各種方式公開分享他們與您的關係。您可以從這些貼文中取消標記，但是隱私不具有追溯性（privacy does not work retroactively）。您可能會想與朋友談談自己願不願意被他們公開分享動態。
